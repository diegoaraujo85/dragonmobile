## 🚀️ Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [x]React Native
- [x]Styled Components
- [x]React Navigation

## 💻 Projeto

Consulta API de Dragon Ball e mostra informações sobre Personagens e Planetas.

## Uso
```bash
git clone https://github.com/diegoaraujo85/dragonmobile.git

cd dragonmobile

yarn install

yarn start

yarn android
```
